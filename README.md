<h1>Cassava and Tomato Disease Detection</h1>
<p>
  This project evaluates three ML models based on three supervised learning algorithms; <br>
  <ol>
    <li><h4>Multiclass Support Vector Machine (SVM)</h4></li>
    <li><h4>Standard Convolutional Neural Network (CNN)</h4></li>
    <li><h4>ResNet50: a classical Convolutional Neural Network</h4></li>
  </ol>
</p>

<p>
  <h3>Libraries used</h3>
  <ol>
    <li><b>SkLearn:</b> For SVM and performance metrics</li>
    <li><b>Pytorch, TorchVision:</b> For CNNs and ResNet50</li>
    <li><b>Splitfolders:</b> For splitting datasets</li>
    <li><b>Matplotlib:</b> For plotting and graphs</li>
    <li><b>PIL:</b> For images</li>
    <li><b>Numpy:</b> For numerical computationss</li>
    <li><b>Pandas:</b> For working with dataframes</li>
  </ol>
</p>
